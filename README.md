Подземелья Древних: Руины Кристального Замка

Консольная roguelike RPG с процедурной генерацией уровней и системой развития персонажа.

Описание

Игрок исследует многоуровневые подземелья, сражается с врагами, собирает сокровища и развивает своего героя. Каждый уровень генерируется случайно, на каждом третьем уровне появляется босс с уникальными способностями.

Запуск игры

Требуется Python 3.8 или выше. Запустите файл:

```
python OOP_RPG.py
```

Управление

· W/A/S/D - движение
· I - открыть инвентарь
· H - пожертвовать 10 HP для увеличения силы
· Q - выход из игры

В бою выбирайте действия цифрами (1-3): атака, лечение, защита/побег.

Как играть

1. Выберите имя героя и уровень сложности (новичок, воин, мастер)
2. Исследуйте подземелья, перемещаясь по комнатам
3. Сражайтесь с врагами: обычными, элитными и боссами
4. Собирайте сокровища для улучшения характеристик
5. Повышайте уровень героя, получая опыт за победы
6. Пройдите 15 уровней, чтобы завершить игру

Особенности

· Процедурная генерация: каждый уровень уникален
· 3 типа врагов с разным поведением
· Боссы с фазами боя и уникальными способностями
· Система развития: повышение уровня улучшает характеристики
· Разные уровни сложности балансируют игру
· Цветной интерфейс в консоли

Архитектура

Проект использует паттерн Abstract Factory для создания врагов и сокровищ в зависимости от выбранной сложности. Основные классы:

· Hero - представляет игрока и врагов
· GameMap - управляет игровым миром и генерацией уровней
· CharacterFactory/TreasureFactory - фабрики для создания объектов
· Room - представляет комнаты в подземелье
